## 解决方案：麻辣龙虾 —— OpenClaw 一键安装助手

**麻辣龙虾（MalaLongxia）** 是一款专为中国用户打造的桌面应用，把 OpenClaw 的整个安装流程变成了「下一步、下一步、完成」。

### 它能帮你做什么？

**1. 环境智能检测**

打开应用，自动检测你的操作系统、Node.js 版本、网络连通性。已经装好的依赖直接跳过，缺什么补什么。

**2. Node.js 自动安装 + 镜像加速**

没装 Node.js？一键安装。版本太低？一键升级。内置阿里云、腾讯云、清华大学等国内镜像源，自动测速选择最快节点，告别龟速下载。

**3. OpenClaw 一键部署**

不用手动 `npm install`，不用记任何命令。点一下按钮，自动完成下载、安装、配置，全程可视化进度条和实时日志。

**4. AI API 可视化配置**

支持多家 AI 供应商（OpenAI、Claude、国内大模型等），图形界面填入 API Key，一键测试连接是否正常。不用再手动编辑 `.env` 文件。

**5. 中英双语界面**

默认中文，一键切换英文，方便国际团队协作。
